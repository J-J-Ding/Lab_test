## 🎨 设计规范

### 配色方案
```css
主色: #0066cc (科技蓝)
深色: #004499
浅色: #4da3ff
辅助色: #00c4cc (青色)
深蓝: #0a1628 (导航栏背景)
```

### 字体规范
- 标题: 36px / 24px / 18px
- 正文: 16px / 14px
- 辅助: 13px / 12px

### 间距规范
- 卡片间距: 25px - 30px
- Section 间距: 60px
- 卡片内边距: 25px - 30px

### 圆角规范
- 导航: 4px
- 卡片: 8px / 12px
- 按钮: 6px / 30px

## 📷 图片占位说明

由于暂无实际图片，建议使用以下占位方式：

1. **直接引用 lab-logo.png**（已有）
2. **使用彩色占位图**
   ```html
   <div style="width:100%;height:180px;background:linear-gradient(135deg,#667eea,#764ba2);display:flex;align-items:center;justify-content:center;color:#fff;font-size:24px;">图片占位</div>
   ```

3. **使用在线占位图服务**
   - https://via.placeholder.com/800x400
   - https://picsum.photos/800/400

## 🔗 作者归属系统

### 如何实现

1. **在内容页面设置作者**
   ```html
   <div class="achievement-detail-author">
     <img src="../../images/zhangsan.jpg" alt="张三" />
     <span>作者：zhangsan</span>
   </div>
   ```

2. **在团队成员页面设置**
   ```javascript
   {
     id: 'zhangsan',
     name: '张三',
     authorName: 'zhangsan',
     ...
   }
   ```

3. **成员详情页自动加载**
   - 根据 authorName 过滤所有栏目
   - 显示该成员的所有成果

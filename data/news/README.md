# 新闻栏目操作手册

## 📁 目录结构

```
news/
├── index.html                  # 新闻列表页
├── css/
│   └── news_detail.css      # 新闻详情页样式
└── 新闻文件夹/              # 每条新闻一个文件夹
    ├── 封面图.png          # 新闻封面图
    ├── index.html          # 新闻详情页
    └── 附件1.xxx          # 可选附件
    └── 附件2.xxx
```

---

## 📝 添加新新闻

### 步骤 1：创建新闻文件夹

1. 在 `news/` 目录下创建新文件夹
2. **命名规则**：`日期+新闻标题`（不含特殊字符）
   - 示例：`20251215人工智能与自动化学院关于开展2026年硕博连读工作的安排`

### 步骤 2：准备新闻资料

- **封面图**：命名为 `封面图.png`，建议尺寸 800x400px
- **附件（可选）**：命名为 `附件1-文件名.扩展名`，`附件2-文件名.扩展名`...

### 步骤 3：创建新闻详情页

复制已有模板文件 `index.html` 到新创建的文件夹中
---

## 🔗 添加附件

### 格式要求

- 文件命名：`附件N-描述性文件名.扩展名`
  - 示例：`附件1-硕博连读申请系统操作指南.docx`
  - 示例：`附件2-《攻读博士学位期间拟开展的研究计划》模板.doc`

### 支持的文件类型

- 📄 Word 文档：`.docx`, `.doc`
- 📊 Excel 表格：`.xlsx`, `.xls`
- 📄 PDF 文档：`.pdf`
- 🖼️ 图片：`.png`, `.jpg`, `.jpeg`

---

## 📄 添加封面图

### 图片要求

- **文件名**：必须为 `封面图.png`
- **建议尺寸**：800x400px 或 1200x600px
- **格式**：PNG（推荐）或 JPG

---

## ✏️ 内容格式说明

### 正文结构

```html
<div class="news-content">
  <h3>标题</h3>
  段落内容...<br>
  <strong>重点内容</strong><br>
</div>
```

### 常用标签

- `<h3>` - 标题
- `<strong>` - 加粗强调
- `<br>` - 换行

---

## 🚀 快速开始

1. 创建新闻文件夹（日期+标题）
2. 准备封面图和附件
3. 复制 HTML 模板并填充内容
4. 完成！

---

## ⚠️ 注意事项

1. **文件夹命名**：使用 `日期+标题` 格式，便于排序
2. **图片路径**：封面图必须命名为 `封面图.png`
3. **附件命名**：使用 `附件N-描述.扩展名` 格式
4. **日期格式**：使用 `YYYY-MM-DD` 格式（如 2025-12-15）
5. **相对路径**：CSS 路径为 `../css/news_detail.css`

---

## 添加新闻列表页

### 列表页要求

- 文件名：`index.html`
- 位置：新闻文件夹根目录
- 内容：包含新闻列表的HTML文件

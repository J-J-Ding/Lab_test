# 成员去向栏目内容添加指南

## 📝 概述

本指南帮助实验室成员快速、规范地添加毕业生去向信息到网站。每个毕业生需要添加一个独立的卡片，展示个人信息、学术成果和就业去向。

---

## 📂 文件结构

```
data/graduates/
├── index.html                    # 毕业生列表页（已存在）
└── figs/                        # 毕业生照片文件夹（已存在）
    ├── 陈悦江.jpg               # 毕业生照片
    ├── 张三.jpg                 # 其他毕业生照片
    └── 李四.jpg
```

---

## 🚀 快速开始（2步完成）

### 第1步：准备毕业生照片

将毕业生的照片放到 `data/graduates/figs/` 目录下：

**照片要求：**
- 格式：JPG 或 PNG
- 尺寸：建议 200×200px 或 400×400px（正方形最佳）
- 文件名：使用毕业生姓名，如 `张三.jpg`、`陈悦江.jpg`
- 内容：清晰的个人照或证件照

### 第2步：添加毕业生卡片

在 `data/graduates/index.html` 中添加毕业生卡片。

## 📋 详细修改说明

### 1. 照片路径（第172行）

```html
<img src="figs/张三.jpg" alt="张三" />
```

**注意事项：**
- 照片必须放在 `data/graduates/figs/` 目录下
- 文件名与 `alt` 属性保持一致（用于无障碍访问）
- 如果没有照片，可以留空或使用占位符

### 2. 毕业生姓名（第175行）

```html
<div class="item-title">张三</div>
```

**格式：** 使用全名

### 3. 毕业信息（第177-179行）

```html
<span>📅 2025届</span>
<span>🎓 硕士学位</span>
<span>🔬 深度学习</span>
```

**字段说明：**

| 字段 | 说明 | 可选值 |
|-----|------|--------|
| 📅 毕业届次 | 如 2025届、2024届 | 根据实际年份 |
| 🎓 学位 | 学位类型 | 硕士学位、博士学位 |
| 🔬 研究方向 | 主要研究领域 | 深度学习、机器学习、计算机视觉、自然语言处理、大数据、强化学习等 |

### 4. 个人简介（第181-183行）

```html
<div class="item-content">
  <p>张三同学于2022年加入智能系统实验室，师从王燕舞教授攻读硕士学位。在读期间，主要从事深度学习与时间序列预测相关研究，参与多项国家级项目，发表高水平论文2篇，授权发明专利1项。</p>
</div>
```

**写作结构：**
- **入学时间与导师**：哪年入学、师从哪位老师
- **研究方向**：主要研究什么内容
- **学术成果**：发表论文、申请专利、参与项目、获奖情况

**写作要点：**
- 使用第三人称（"张三同学"）
- 简洁明了，控制在100-200字
- 突出主要成果和亮点

**示例：**
> 李四同学于2021年加入智能系统实验室，师从王燕舞教授攻读硕士学位。在读期间，主要从事计算机视觉与目标检测相关研究，参与国家级项目1项、省部级项目2项，发表高水平论文1篇，获得校级奖学金3次。

### 5. 学术成果统计（第185-196行）

```html
<div class="item-info">
  <div class="info-item">
    <span class="info-icon">📄</span>
    <span class="info-label">论文：2篇</span>
  </div>
  <div class="info-item">
    <span class="info-icon">🔬</span>
    <span class="info-label">专利：1项</span>
  </div>
  <div class="info-item">
    <span class="info-icon">🏆</span>
    <span class="info-label">获奖：3项</span>
  </div>
</div>
```

**可添加的成果类型：**

| 图标 | 类型 | 说明 |
|-----|------|------|
| 📄 | 论文 | 发表的学术论文数量 |
| 🔬 | 专利 | 授权的发明专利数量 |
| 🏆 | 获奖 | 获得的奖项数量 |
| 🎓 | 奖学金 | 获得的奖学金次数 |
| 📊 | 项目 | 参与的项目数量 |
| 🏢 | 实习 | 实习经历（可选） |

**数值格式：**
- 使用阿拉伯数字，如 `2篇`、`1项`、`3次`
- 如果为0，可以填写 `无` 或省略该项

### 6. 就业去向（第199-203行）


**字段说明：**

| 字段 | 说明 | 示例 |
|-----|------|------|
| 就业单位 | 工作单位名称 | 腾讯科技有限公司、阿里巴巴、字节跳动、南方电网、华为等 |
| 职位标签 | 职位名称 | 算法工程师、数据科学家、软件开发工程师、研究员等 |

**就业去向类型：**
- **互联网公司**：腾讯、阿里、字节、百度、美团等
- **科技企业**：华为、中兴、小米、OPPO等
- **金融行业**：工商银行、建设银行、招商银行、支付宝等
- **国企央企**：南方电网、国家电网、中国电信等
- **科研院所**：中科院、高校、研究院所等
- **继续深造**：国内外知名高校读博

**职位类型：**
- **算法类**：算法工程师、数据科学家、机器学习工程师
- **开发类**：软件开发工程师、前端工程师、后端工程师
- **产品类**：产品经理、产品运营
- **研究类**：研究员、博士后、助理教授

## 🔄 添加毕业生卡片的完整流程

### 步骤1：准备照片
- 准备毕业生的个人照片（正方形最佳）
- 将照片重命名为 `姓名.jpg`（如 `张三.jpg`）
- 将照片放到 `data/graduates/figs/` 目录下

### 步骤2：添加卡片
- 打开 `data/graduates/index.html`
- 找到 `<div id="contentList">` 容器
- 在合适的位置添加新的毕业生卡片

### 步骤3：修改内容
按照模板修改以下内容：
- [ ] 照片路径
- [ ] 毕业生姓名
- [ ] 毕业届次、学位、研究方向
- [ ] 个人简介
- [ ] 学术成果统计
- [ ] 就业单位和职位

### 步骤4：按届次排序
- 按毕业届次降序排列（最新的在最上面）
- 同届毕业生可以按姓名拼音排序
